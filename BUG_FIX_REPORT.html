Technical Details — OTP Timer Fix

  • Stores endTimeRef = Date.now() + RESEND_TIMER * 1000 as the target timestamp.
  • Each tick calculates remaining = Math.max(0, (endTime - Date.now()) / 1000), so even after the screen goes off, the countdown is accurate on resume.
  • AppState.addEventListener('change', ...) triggers an immediate tick when the app returns to the foreground.
  • On resend, endTimeRef is reset to a new future timestamp.

Technical Details — Attachment Fix

  • Base64 image data is written to FileSystem.cacheDirectory as a temporary file.
  • Sharing.shareAsync(fileUri) opens the native share/preview dialog, allowing the user to view, save, or share the attachment.
  • This approach works on both iOS and Android, unlike data: URI schemes which are unsupported by Linking.openURL on mobile.
