1. Project Overview

1.1 Introduction

AcePick is a comprehensive mobile application built with React Native (Expo) that connects clients with skilled professionals (artisans). The platform facilitates service booking, product marketplace transactions, and delivery logistics.

1.2 Technology Stack

Component Technology Version
Framework React Native (Expo) SDK 52
Language TypeScript 5.x
State Management Redux Toolkit + Redux Persist Latest
Navigation Expo Router Latest
Styling NativeWind (TailwindCSS) 4.x
API Client Axios + React Query Latest
Real-time Socket.io Client Latest
Voice Calls WebRTC Latest

1.3 Backend API

Base URL: https://www.acepickdev.com

Socket Path: /chat

API Timeout: 30,000ms

2. System Architecture

2.1 Application Layers

Layer Description Location
Presentation UI Components, Screens /component, /pages
Business Logic Custom Hooks, Context /hooks, /context
State Management Redux Slices, Store /redux
Data Access API Services /services
Utilities Helpers, Constants /utilizes

2.2 State Management

The application uses Redux Toolkit with the following slices:

  • authSlice: User authentication, tokens, profile
  • chatSlice: Messages, rooms, previous chats
  • contactSlice: Contact list management
  • registerSlice: Registration flow data

6. Development Setup

6.1 Prerequisites

  • Node.js 18+ (LTS recommended)
  • npm or yarn package manager
  • Expo CLI
  • iOS Simulator (Mac) or Android Emulator
  • Expo Go app (for physical device testing)

6.2 Installation

# Clone repository
git clone [repository-url]
cd acepick

# Install dependencies
npm install

# Start development server
npx expo start

6.3 Running the App

# Start with options
npx expo start

# Platform-specific
npx expo start --ios        # iOS Simulator
npx expo start --android    # Android Emulator
npx expo start --web        # Web Browser

# Development commands
Press 'w' - Open web
Press 'i' - Open iOS Simulator
Press 'a' - Open Android Emulator
Press 'r' - Reload app
Press 'm' - Toggle menu

6.4 Building for Production

# Configure EAS Build
eas build:configure

# Build for iOS
eas build --platform ios

# Build for Android
eas build --platform android

# Build APK (Android)
eas build --platform android --profile apk
